1. 멤버 관리

정기적으로 참여하는 회원을 등록하고 관리하는 메뉴입니다.

  • 멤버 추가 : 여자/남자 섹션의 [+] 버튼을 클릭하여 이름과 성별을 입력합니다.
  • 멤버 수정/삭제 : 각 멤버 카드의 [...] 버튼을 클릭하면 수정 또는 삭제할 수 있습니다.
  • 등록된 멤버 정보는 브라우저에 자동 저장됩니다.

2. 대진표 작성

STEP 1 - 게임 날짜 선택

상단의 캘린더에서 게임 날짜를 지정합니다.

STEP 2 - 참여 멤버 선택

  • 좌측 '등록 멤버'에서 카드를 클릭(호버 시 + 표시)하면 우측 '참가 멤버'로 이동합니다.
  • [전체 추가] 버튼으로 모든 멤버를 한 번에 추가할 수 있습니다.
  • 우측 참가 멤버 카드를 클릭(호버 시 - 표시)하면 제거됩니다.
  • 해당 날짜에만 참여하는 게스트는 우측의 [+] 버튼으로 추가합니다.
  • 선택이 완료되면 [멤버 확정]을 클릭합니다.

STEP 3 - 대진 생성

  • 혼복 게임 수를 설정합니다 (0이면 여복/남복만 구성).
  • 알고리즘 옵션을 선택합니다:
    • 페어 우선 : 같은 팀이 되는 페어 횟수를 균등하게
    • 상대 우선 : 상대팀으로 만나는 횟수를 균등하게
    • 히스토리 우선 : 과거 저장된 대진표까지 고려하여 균등하게
    • 무작위 : 랜덤 배치
  • [대진 생성]을 클릭하면 자동으로 대진이 구성됩니다.
  • 게임 수는 항상 최대한 균등하게 배분되며, 게스트의 게임 수가 우선 보장됩니다.

STEP 4 - 대진 편집

  • 상단 멤버 카드를 드래그&드롭하여 특정 슬롯에 배치할 수 있습니다.
  • 슬롯 내 카드끼리도 드래그&드롭으로 교환 가능합니다.
  • 슬롯 카드의 [...] 메뉴에서:
    • Lock : 해당 슬롯을 고정합니다. [대진 생성] 시에도 변경되지 않습니다.
    • 취소 : 슬롯에서 제거합니다.
  • 게임 제목(*1게임* 등)을 드래그하면 게임 순서를 통째로 교환할 수 있습니다.
  • 한 게임에 동일 인물이 중복 배치되면 해당 게임이 빨간색으로 표시됩니다.

STEP 5 - 저장 및 출력

  • [저장하기] : 대진표를 브라우저에 저장합니다. 히스토리에서 불러올 수 있습니다.
  • [이미지 저장] : 대진표를 JPG 이미지 파일로 다운로드합니다.

3. 우측 실시간 통계

  • 게임 수 / 대기 수 : 각 멤버의 출전 게임 수와 대기 횟수
  • 페어 횟수 매트릭스 : 같은 팀으로 배치된 횟수 (색이 진할수록 많음)
  • 상대 횟수 매트릭스 : 상대팀으로 만난 횟수 (색이 진할수록 많음)
  • 대진을 수정할 때마다 실시간으로 업데이트됩니다.

4. 히스토리

  • 캘린더에서 저장된 게임 날짜에 녹색 배지가 표시됩니다.
  • 날짜를 클릭하면 저장된 대진표를 확인할 수 있습니다.
  • [수정하기]를 클릭하면 해당 대진표를 대진표 작성 화면으로 불러옵니다.
  • 우측에는 모든 저장된 대진표를 합산한 누적 페어/상대 매트릭스가 표시됩니다.

5. 게스트 관리

  • 캘린더에서 날짜를 선택한 후 해당 날짜에 참여할 게스트를 남녀 구분하여 등록합니다.
  • 각 게스트에 메모를 작성할 수 있습니다 (카드의 [...] 메뉴).
  • 게스트를 삭제하면 히스토리에서도 즉시 반영됩니다.
  • 우측 게스트 히스토리에서 참여 횟수순으로 모든 게스트 목록을 확인할 수 있습니다.
  • 게스트 이름을 클릭하면 참여 일자를 볼 수 있습니다.
  • 대진표 작성 시 [게스트 불러오기] 버튼으로 해당 날짜에 등록된 게스트를 자동으로 참가 멤버에 추가할 수 있습니다.

6. 게시판

  • 우측의 글 작성 폼에서 제목, 작성자, 본문을 입력하고 저장합니다.
  • 좌측 목록에서 글을 클릭하면 본문이 펼쳐집니다.
  • 본문 하단에서 댓글을 작성할 수 있습니다 (이름 + 내용).
  • 댓글의 [답글] 버튼을 클릭하면 대댓글을 작성할 수 있습니다.

7. 코트 유형 색상 안내

남복남자 4명으로 구성
여복여자 4명으로 구성
혼복남녀 혼합 (각 팀 남1 + 여1)
잡복위 3가지에 해당하지 않는 구성 (최소화됨)

참고 사항

  • 모든 데이터는 Airtable 클라우드에 자동 저장됩니다. 누가 접속하든 동일한 데이터를 볼 수 있습니다.
  • 페이지를 열면 최신 데이터가 자동으로 로드됩니다.
  • 2코트 기준 5게임으로 구성되며, 참가 인원에 따라 대기자가 배정됩니다.
